# PDF Tools (Industry-Ready)

A clean, modern PDF utility app built with **Streamlit**.

## ✅ Features
- **PDF → Images (JPG)**: page ranges, password support, naming rules, preview before convert
- **Images → PDF**: A4/Letter/Auto sizing, fit mode, margins
- **Light/Dark theme toggle**

## ▶️ Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Privacy
- Files are processed **in memory** within your Streamlit session.
- No files are uploaded anywhere unless you deploy this app on a server.
